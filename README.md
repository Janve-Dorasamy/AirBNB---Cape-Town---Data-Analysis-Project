# AirBNB---Cape-Town---Data-Analysis-Project
A project using real world stats from AirBnB listings in Cape Town, using MYSQL, Python, and PowerBI

This project demonstrates the process of cleaning, standardizing, and normalizing a real-world dataset of Airbnb listings in Cape Town. The raw data contained inconsistencies, missing values, and inconsistent data types, causing the raw data to be unusable for data analysis. 

Key Challenges
•	Null values in important columns: price, reviews_per_month, last_review
•	Incorrect data types 
•	Empty columns 
•	Need to normalize repeated information
